###A aplicação tem como objetivo adicionar, buscar, deletar e atualizar livros em um banco de dados;

###Pré-requisitos

Antes de começar, voce vai precisar ter o [NODE.JS](https://nodejs.org/en/) e o [Git](https://git-scm.com) instalado em sua maquina;
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com)
Contém dependencias com mongodb, use o mongodb atlas para a criação do banco [MONGODB](https://www.mongodb.com/);

###Procedimentos

#Clone esse repositório
$git clone https://github.com/lucascota-ld/BookDepository.git

#Substitua as tags '<>' no arquivo index.js na linha 27 pelos seus respectivos dados para realizar a conexão om a banco;

#Instale as dependencias;
$ npm install

#Execute a aplicação
$ npm start

#O seridor iniciará na porta 3000 - acesse - <http://localhost:3000>
$ Teste a api com insominia https://insomnia.rest/download
